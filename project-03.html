[Mobile Web] mgoon Mobiel Web ver.mymgoon

2012년 2월 엠군 모바일 웹 디자인 진행.
엠군 모바일 웹에서는 엠군 동영상 / 마이엠군 / 제휴서비스 등의 바로가기 리스트형태 로 엠군의 주력 서비스들을 바로 이용하기 쉽게 디자인.

마이엠군의 경우 개인 동영상을 SNS 형식과 접목하여 런칭된 서비스이다.
모바일웹을 하면서 제일 힘들었던 것은 웹에서 보여지는 것들을 모바일웹에서도 모두 보여져야하고 액션이 취해여야 했기에 페이지 수며 각 이동경로 등 많이 복잡했고 페이지수도 많아서 조금 고생스러운 작업이였다.
